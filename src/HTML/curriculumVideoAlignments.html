curriculumVideoAlignments endpoint

Overview

The curriculumVideoAlignments endpoint gets info on curriculumVideoAlignments in Prodigy, which is data based off of videos that assist with learning certain educational topics. Seems to require an API key to access, so a bit pointless other then for informational purposes.

Endpoint:

http://localhost:4000/v1/curriculumVideoAlignments/

Example request:

fetch('http://localhost:4000/v1/curriculumVideoAlignments/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

cva[cvaID].ID
cva[cvaID].assetID
cva[cvaID].type
cva[cvaID].gender
cva[cvaID].data
cva[cvaID].metadata
cva[cvaID].name

Data properties:

cva[cvaID].data.name
cva[cvaID].data.expectations

There is only one object in CVA, which is

http://localhost:4000/v1/curriculumVideoAlignments/0

Which is the same as the default endpoint, so there is no need to simplify.
