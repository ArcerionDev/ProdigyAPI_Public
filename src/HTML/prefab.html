Prefab endpoint

The Prefab endpoint seems to have info on all of the settings that make up the UI and the entire game, basically.

Endpoint:

http://localhost:4000/v1/prefab

Example request:

fetch('http://localhost:4000/v1/prefab/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

prefab[prefabID].ID
prefab[prefabID].assetID
prefab[prefabID].type
prefab[prefabID].gender
prefab[prefabID].data
prefab[prefabID].metadata
prefab[prefabID].name

Data properties:

prefab[prefabID].data.name
prefab[prefabID].data.prefab

Get more specific results using the Prefab ID in the API endpoint.

http://localhost:4000/v1/prefab/ID/

Example request:

fetch('http://localhost:4000/v1/prefab/0/').then(response => response.json()).then(data => {console.log(data)})
