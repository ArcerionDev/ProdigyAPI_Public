Dialogue endpoint

Overview

The Dialogue endpoint gets info on dialogue in Prodigy, which shows which characters say which phrases. It's mostly informational for Prodigy, as the data doesn't actually have the dialogue in it.

Endpoint:

http://localhost:4000/v1/dialogue/

Example request:

fetch('http://localhost:4000/v1/dialogue/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

dialogue[dialogueID].ID
dialogue[dialogueID].assetID
dialogue[dialogueID].type
dialogue[dialogueID].gender
dialogue[dialogueID].data
dialogue[dialogueID].metadata
dialogue[dialogueID].name

Data properties:

dialogue[dialogueID].data.audio
dialogue[dialogueID].data.avatar
dialogue[dialogueID].data.textKey

Get more specific results using the dialogue ID in the API endpoint.

http://localhost:4000/v1/dialogue/ID/

Example request:

fetch('http://localhost:4000/v1/dialogue/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":10,"assetID":3660,"type":"dialogue","gender":0,"data":{"audio":{"d":1,"s":0,"tag":"voice-10-12"},"avatar":{"atlas":"atlas-50","frameName":"face_2","animationMod":"4","spineAttachment":""},"textKey":"DIALOGUE_NOOT_0"},"metadata":{},"name":"dialogue_noot_0"}
