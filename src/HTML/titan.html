Titan endpoint

The Titan endpoint gives info on the various Titans that appear across the map.

Endpoint:

http://localhost:4000/v1/titan

Example request:

fetch('http://localhost:4000/v1/titan/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

titan[titanID].ID
titan[titanID].assetID
titan[titanID].type
titan[titanID].gender
titan[titanID].data
titan[titanID].metadata
titan[titanID].name

Data properties:

titan[titanID].data.name
titan[titanID].data.zone
titan[titanID].data.maxHealth
titan[titanID].data.element
titan[titanID].data.difficulty
titan[titanID].data.spellList
titan[titanID].data.rewards
titan[titanID].data.spine

Get more specific results using the Titan ID in the API endpoint.

http://localhost:4000/v1/titan/ID/

Example request:

fetch('http://localhost:4000/v1/titan/0/').then(response => response.json()).then(data => {console.log(data)})
