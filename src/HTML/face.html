Face endpoint

The Face endpoint gives info on all of the faces you get obtain in Prodigy.

Endpoint:

http://localhost:4000/v1/face

Example request:

fetch('http://localhost:4000/v1/face/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

face[faceID].ID
face[faceID].assetID
face[faceID].type
face[faceID].gender
face[faceID].data
face[faceID].metadata
face[faceID].name

Data properties:

face[faceID].data.price
face[faceID].data.name
face[faceID].data.flavorText

Get more specific results using the Face ID in the API endpoint.

http://localhost:4000/v1/face/ID/

Example request:

fetch('http://localhost:4000/v1/face/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":365,"type":"face","gender":1,"data":{"price":250,"name":"Lively","flavorText":"Buying this will change your face style!"},"metadata":{"type":"singleImage","v":4,"vIcon":2},"name":""}
