Outfit endpoint

The Outfit endpoint gives info on the robes you can get ingame.

Endpoint:

http://localhost:4000/v1/outfit

Example request:

fetch('http://localhost:4000/v1/outfit/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

outfit[outfitID].ID
outfit[outfitID].assetID
outfit[outfitID].type
outfit[outfitID].gender
outfit[outfitID].data
outfit[outfitID].metadata
outfit[outfitID].name

Data properties:

outfit[outfitID].data.drop
outfit[outfitID].data.name
outfit[outfitID].data.price
outfit[outfitID].data.member
outfit[outfitID].data.rarity
outfit[outfitID].data.effects
outfit[outfitID].data.flavorText

Get more specific results using the Outfit ID in the API endpoint.

http://localhost:4000/v1/outfit/ID/

Example request:

fetch('http://localhost:4000/v1/outfit/0/').then(response => response.json()).then(data => {console.log(data)})
