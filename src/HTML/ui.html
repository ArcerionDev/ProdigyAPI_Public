Ui endpoint

The Ui endpoint gives info on the user interface (like buttons and your bottom tab with kennel, mail, etc.)

Endpoint:

http://localhost:4000/v1/ui

Example request:

fetch('http://localhost:4000/v1/ui/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

ui[uiID].ID
ui[uiID].assetID
ui[uiID].type
ui[uiID].gender
ui[uiID].data
ui[uiID].metadata
ui[uiID].name

Data properties:

ui[uiID].data.name
ui[uiID].data.elements
ui[uiID].data.templates

Get more specific results using the Ui ID in the API endpoint.

http://localhost:4000/v1/ui/ID/

Example request:

fetch('http://localhost:4000/v1/ui/0/').then(response => response.json()).then(data => {console.log(data)})
