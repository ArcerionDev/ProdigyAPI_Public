Key endpoint

The Key endpoint has info on all of the key items in Prodigy.

Endpoint:

http://localhost:4000/v1/key

Example request:

fetch('http://localhost:4000/v1/key/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

key[keyID].ID
key[keyID].assetID
key[keyID].type
key[keyID].gender
key[keyID].data
key[keyID].metadata
key[keyID].name

Data properties:

key[keyID].data.name
key[keyID].data.levels
key[keyID].data.flavorText

Get more specific results using the Key ID in the API endpoint.

http://localhost:4000/v1/key/ID/

Example request:

fetch('http://localhost:4000/v1/key/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":567,"type":"key","gender":0,"data":{"name":"Dyno Dig Shovel","levels":3,"flavorText":"Use this special shovel to play the Dyno Dig game where you can dig up Dyno fossils to fill the museum."},"metadata":{"vIcon":2,"v":1,"type":"singleImage","subType":0},"name":"Shovel"}
