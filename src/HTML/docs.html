Prodigy API Docs

Overview

The Prodigy API is an open API for getting info on Prodigy data. All info is stored in an array.

Authentication

No Authentication is required.

Ratelimiting

You can make up to 500 requests an hour.

Status Codes

+ 200 OK
- 404 Not Found

Note

When you specify an ID for an object, the ID starts at 0 in the URL.
