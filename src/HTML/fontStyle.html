FontStyle endpoint

The FontStyle endpoint gives info on fonts in Prodigy.

Endpoint:

http://localhost:4000/v1/fontStyle

Example request:

fetch('http://localhost:4000/v1/fontStyle/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

fontStyle[fontStyleID].ID
fontStyle[fontStyleID].assetID
fontStyle[fontStyleID].type
fontStyle[fontStyleID].gender
fontStyle[fontStyleID].data
fontStyle[fontStyleID].metadata
fontStyle[fontStyleID].name

Data properties:

fontStyle[fontStyleID].data.font
fontStyle[fontStyleID].data.fontSize
fontStyle[fontStyleID].data.fontWeight
fontStyle[fontStyleID].data.smoothed
fontStyle[fontStyleID].data.fill
fontStyle[fontStyleID].data.blendMode
fontStyle[fontStyleID].data.padding

Get more specific results using the FontStyle ID in the API endpoint.

http://localhost:4000/v1/fontStyle/ID/

Example request:

fetch('http://localhost:4000/v1/fontStyle/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":2126,"type":"fontStyle","gender":0,"data":{"font":"Fredoka One","fontSize":18,"fontWeight":"normal","smoothed":false,"fill":"#FF0000","blendMode":0,"padding":{"x":10,"y":0}},"metadata":{},"name":"Fredoka One"}
