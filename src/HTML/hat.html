Hat endpoint

The Hat endpoint gives info on all of the hats you can collect and purchase in the game.

Endpoint:

http://localhost:4000/v1/hat

Example request:

fetch('http://localhost:4000/v1/hat/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

hat[hatID].ID
hat[hatID].assetID
hat[hatID].type
hat[hatID].gender
hat[hatID].data
hat[hatID].metadata
hat[hatID].name

Data properties:

hat[hatID].data.drop
hat[hatID].data.name
hat[hatID].data.type
hat[hatID].data.price
hat[hatID].data.member
hat[hatID].data.rarity
hat[hatID].data.effects
hat[hatID].data.flavorText

Get more specific results using the Hat ID in the API endpoint.

http://localhost:4000/v1/hat/ID/

Example request:

fetch('http://localhost:4000/v1/hat/0/').then(response => response.json()).then(data => {console.log(data)})
