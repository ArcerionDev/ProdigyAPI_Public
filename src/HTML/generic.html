Generic endpoint

The Generic endpoint gives a bunch of startup data, very unclear, because there are a ton of variations and they don't all fit into the same category...

Endpoint:

http://localhost:4000/v1/generic

Example request:

fetch('http://localhost:4000/v1/generic/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

generic[genericID].ID
generic[genericID].assetID
generic[genericID].type
generic[genericID].gender
generic[genericID].data
generic[genericID].metadata
generic[genericID].name

Data properties:

generic[genericID].data.name
generic[genericID].data.bountyPetIDs
generic[genericID].data.bountyDurationSeconds
generic[genericID].data.tiers

Get more specific results using the Generic ID in the API endpoint.

http://localhost:4000/v1/generic/ID/

Example request:

fetch('http://localhost:4000/v1/generic/0/').then(response => response.json()).then(data => {console.log(data)})
