PrizeWheel endpoint

The PrizeWheel endpoint gives info on the assets for the Wheel of Wonder, and the rewards you can get from it.

Endpoint:

http://localhost:4000/v1/prizeWheel

Example request:

fetch('http://localhost:4000/v1/prizeWheel/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

prizeWheel[prizeWheelID].ID
prizeWheel[prizeWheelID].assetID
prizeWheel[prizeWheelID].type
prizeWheel[prizeWheelID].gender
prizeWheel[prizeWheelID].data
prizeWheel[prizeWheelID].metadata
prizeWheel[prizeWheelID].name

prizeWheel[prizeWheelID].data.itemPools

prizeWheel[prizeWheelID].data.eventType
