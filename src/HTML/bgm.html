BGM endpoint

Overview

THe BGM endpoint is mostly informational to Prodigy, so it can tell which background music is which. Not much use.

Endpoint:

http://localhost:4000/v1/bgm/

Example request:

fetch('http://localhost:4000/v1/bgm/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

bgms[bgmID].ID
bgms[bgmID].assetID
bgms[bgmID].type
bgms[bgmID].gender
bgms[bgmID].data
bgms[bgmID].metadata
bgms[bgmID].name

Data properties:

bgms[bgmID].data.filename

Get more specific results using the atlas ID in the API endpoint.

http://localhost:4000/v1/bgm/ID/

Example request:

fetch('http://localhost:4000/v1/bgm/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":1806,"type":"bgm","gender":0,"data":{"filename":"bgm-victory"},"metadata":{"assets":{"default":{"v":2}}},"name":"bgm-victory"}
