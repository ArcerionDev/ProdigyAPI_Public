Dorm endpoint

The Dorm endpoint gives info on house items you can get ingame.

Endpoint:

http://localhost:4000/v1/dorm

Example request:

fetch('http://localhost:4000/v1/dorm/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

dorm[dormID].ID
dorm[dormID].assetID
dorm[dormID].type
dorm[dormID].gender
dorm[dormID].data
dorm[dormID].metadata
dorm[dormID].name

Data properties:

dorm[dormID].data.r
dorm[dormID].data.name
dorm[dormID].data.price
dorm[dormID].data.member
dorm[dormID].data.rarity
dorm[dormID].data.category
dorm[dormID].data.flavorText

Get more specific results using the Dorm ID in the API endpoint.

http://localhost:4000/v1/dorm/ID/

Example request:

fetch('http://localhost:4000/v1/dorm/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":614,"type":"dorm","gender":0,"data":{"r":4,"name":"Comfy Chair","price":50,"member":0,"rarity":0,"category":"Comfy","flavorText":"Just a regular old chair.
