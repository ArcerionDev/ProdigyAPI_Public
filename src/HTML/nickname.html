Nickname endpoint

The Nickname endpoint gives info on nicknames you can get ingame (e.g. [NAME] the W12-4RD).

Endpoint:

http://localhost:4000/v1/nickname

Example request:

fetch('http://localhost:4000/v1/nickname/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

nickname[nicknameID].ID
nickname[nicknameID].assetID
nickname[nicknameID].type
nickname[nicknameID].gender
nickname[nicknameID].data
nickname[nicknameID].metadata
nickname[nicknameID].name

Data properties:

nickname[nicknameID].data.value

Get more specific results using the Nickname ID in the API endpoint.

http://localhost:4000/v1/nickname/ID/

Example request:

fetch('http://localhost:4000/v1/nickname/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":3094,"type":"nickname","gender":0,"data":{"value":"Buccaneer {first}"},"metadata":{},"name":"Buccaneer"}
