Hair endpoint

The Hair endpoint gives info on all of the hair in the game.

Endpoint:

http://localhost:4000/v1/hair

Example request:

fetch('http://localhost:4000/v1/hair/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

hair[hairID].ID
hair[hairID].assetID
hair[hairID].type
hair[hairID].gender
hair[hairID].data
hair[hairID].metadata
hair[hairID].name

Data properties:

hair[hairID].data.name
hair[hairID].data.price
hair[hairID].data.member
hair[hairID].data.flavorText

Get more specific results using the Hair ID in the API endpoint.

http://localhost:4000/v1/hair/ID/

Example request:

fetch('http://localhost:4000/v1/hair/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":285,"type":"hair","gender":1,"data":{"name":"Grace","price":250,"member":0,"flavorText":"Buying this will change your hair style!"},"metadata":{"x":-6,"y":-37,"pivot":{"x":64,"y":36},"vIcon":12,"type":"singleImage","v":5},"name":"Grace"}
