Gender endpoint

The Gender endpoint gives info on the genders you can change to in Prodigy.

Endpoint:

http://localhost:4000/v1/gender

Example request:

fetch('http://localhost:4000/v1/gender/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

gender[genderID].ID
gender[genderID].assetID
gender[genderID].type
gender[genderID].gender
gender[genderID].data
gender[genderID].metadata
gender[genderID].name

Data properties:

gender[genderID].data.member
gender[genderID].data.name
gender[genderID].data.saveDataID
gender[genderID].data.cost
gender[genderID].data.flavorText

Get more specific results using the Gender ID in the API endpoint.

http://localhost:4000/v1/gender/ID/

Example request:

fetch('http://localhost:4000/v1/gender/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":1195,"type":"gender","gender":0,"data":{"member":0,"name":"Boy","saveDataID":"male","cost":[{"type":"currency","ID":1,"N":250}],"flavorText":"Buying this will change your gender!"},"metadata":{"v":1,"type":"singleImage","vIcon":1},"name":"Male"}
