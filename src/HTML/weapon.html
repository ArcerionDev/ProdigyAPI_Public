Weapon endpoint

The Weapon endpoint gives info on all of the wands you can get in the game.

Endpoint:

http://localhost:4000/v1/weapon

Example request:

fetch('http://localhost:4000/v1/weapon/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

weapon[weaponID].ID
weapon[weaponID].assetID
weapon[weaponID].type
weapon[weaponID].gender
weapon[weaponID].data
weapon[weaponID].metadata
weapon[weaponID].name

Data properties:

weapon[weaponID].data.drop
weapon[weaponID].data.name
weapon[weaponID].data.price
weapon[weaponID].data.member
weapon[weaponID].data.rarity
weapon[weaponID].data.effects
weapon[weaponID].data.flavorText

Get more specific results using the Weapon ID in the API endpoint.

http://localhost:4000/v1/weapon/ID/

Example request:

fetch('http://localhost:4000/v1/weapon/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":111,"type":"weapon","gender":0,"data":{"drop":1,"name":"Enchanted Stick","price":350,"member":0,"rarity":1,"effects":[30],"flavorText":"The most unassuming of wands, it is said to be imbued with natural energy."},"metadata":{"v":1,"size":0,"type":"singleImage","angle":50,"vIcon":2,"anchor":{"x":0.45,"y":0.85}},"name":"Enchanted Stick"}
