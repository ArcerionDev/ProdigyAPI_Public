Fossil endpoint

The Fossil endpoint gets info on fossils you can get from Dyno Dig Oasis in game.

Endpoint:

http://localhost:4000/v1/fossil

Example request:

fetch('http://localhost:4000/v1/fossil/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

fossil[fossilID].ID
fossil[fossilID].assetID
fossil[fossilID].type
fossil[fossilID].gender
fossil[fossilID].data
fossil[fossilID].metadata
fossil[fossilID].name

Data properties:

fossil[fossilID].data.name
fossil[fossilID].data.count
fossil[fossilID].data.price
fossil[fossilID].data.rarity
fossil[fossilID].data.flavorText

Get more specific results using the Fossil ID in the API endpoint.

http://localhost:4000/v1/fossil/ID/

Example request:

fetch('http://localhost:4000/v1/fossil/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":580,"type":"fossil","gender":0,"data":{"name":"Terrosaur Bone","count":20,"price":50,"rarity":0,"flavorText":"This bone looks like a piece from a much larger skeleton. I better find them all!"},"metadata":{"vIcon":2},"name":"Terrosaur Bone"}
