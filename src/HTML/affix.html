Affix endpoint

Overview

The affix endpoint gets info on potions and potion bonuses in Prodigy, which are found by using potions that are collectable for members.

Endpoint:

http://localhost:4000/v1/affix/

Example request:

fetch('http://localhost:4000/v1/affix/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

affix[affixID].ID
affix[affixID].assetID
affix[affixID].type
affix[affixID].gender
affix[affixID].data
affix[affixID].metadata
affix[affixID].name

Data properties:

affix[affixID].data.label
affix[affixID].data.opType
affix[affixID].data.type
affix[affixID].data.value
affix[affixID].data.name

Get more specific results using the affix ID in the API endpoint.

http://localhost:4000/v1/affix/ID/

Example request:

fetch('http://localhost:4000/v1/affix/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":1059,"type":"affix","gender":0,"data":{"label":"Fire Resist: ","opType":0,"type":"fire-resist","value":-20,"name":"Fire Resist 1"},"metadata":{"v":1,"type":"singleImage","vIcon":1},"name":"Fire Resist 1"}
