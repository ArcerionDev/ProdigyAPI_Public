Ad endpoint

Overview

The ad endpoint gets info on ads in Prodigy, which are found on the Wizard Watch message board.

Endpoint:

http://localhost:4000/v1/ad/

Example request:

fetch('http://localhost:4000/v1/ad/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

ads[adID].ID
ads[adID].assetID
ads[adID].type
ads[adID].gender
ads[adID].data
ads[adID].metadata
ads[adID].name

Data properties (ads[adID].data)

ads[adID].data.name
ads[adID].data.sendDate
ads[adID].data.startDate
ads[adID].data.expiryDate
ads[adID].data.uiID
ads[adID].data.dateTextName
ads[adID].data.buttons
ads[adID].data.assets
ads[adID].data.priority

Get more specific results using the ad ID in the API endpoint.

http://localhost:4000/v1/ad/ID/

Example request:

fetch('http://localhost:4000/v1/ad/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:
