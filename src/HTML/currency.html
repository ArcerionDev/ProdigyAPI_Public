Currency endpoint

Overview

The currency endpoint gets info on currencies you can get in-game.

Endpoint:

http://localhost:4000/v1/currency/

Example request:

fetch('http://localhost:4000/v1/currency/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

currency[currencyID].ID
currency[currencyID].assetID
currency[currencyID].type
currency[currencyID].gender
currency[currencyID].data
currency[currencyID].metadata
currency[currencyID].name

Data properties:

currency[currencyID].data.drop
currency[currencyID].data.name
currency[currencyID].data.price
currency[currencyID].data.modifier
currency[currencyID].data.flavorText
currency[currencyID].data.icon

Get more specific results using the currency ID in the API endpoint.

http://localhost:4000/v1/currency/ID/

Example request:

fetch('http://localhost:4000/v1/currency/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":605,"type":"currency","gender":0,"data":{"drop":0,"name":"Gold","price":50,"modifier":1,"flavorText":"","icon":"small-icon-gold"},"metadata":{"vIcon":10},"name":"Gold"}
