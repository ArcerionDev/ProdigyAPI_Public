GameFeed endpoint

The GameFeed endpoint gives the templates for things that you send in Wizard watch, such as "I ranked up in the achievement: _!"

Endpoint:

http://localhost:4000/v1/gameFeed

Example request:

fetch('http://localhost:4000/v1/gameFeed/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

gameFeed[gameFeedID].ID
gameFeed[gameFeedID].assetID
gameFeed[gameFeedID].type
gameFeed[gameFeedID].gender
gameFeed[gameFeedID].data
gameFeed[gameFeedID].metadata
gameFeed[gameFeedID].name

Data properties:

gameFeed[gameFeedID].data.key
gameFeed[gameFeedID].data.category
gameFeed[gameFeedID].data.notification
gameFeed[gameFeedID].data.modal
gameFeed[gameFeedID].data.feedText
gameFeed[gameFeedID].data.icon
gameFeed[gameFeedID].data.autoShare
gameFeed[gameFeedID].data.eventAction

Get more specific results using the GameFeed ID in the API endpoint.

http://localhost:4000/v1/gameFeed/ID/

Example request:

fetch('http://localhost:4000/v1/gameFeed/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":3205,"type":"gameFeed","gender":0,"data":{"key":"achievement","category":"award","notification":{"self":"You ranked up in the '${mapping[0]}' Achievement!","friend":"${userName} ranked up in an Achievement!"},"modal":{"title":"","body":"","action":""},"feedText":"I ranked up in the achievement: '${mapping[0]}'!","icon":"","autoShare":false,"eventAction":""},"metadata":{},"name":"achievement"}
