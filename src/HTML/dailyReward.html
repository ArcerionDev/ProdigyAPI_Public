dailyReward endpoint

Overview

The dailyReward endpoint gets info on daily rewards in Prodigy, which are the possible rewards that you can get for answering a certain amount of questions, depending on your education plan.

Endpoint:

http://localhost:4000/v1/dailyReward/

Example request:

fetch('http://localhost:4000/v1/dailyReward/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

dailyReward[dailyRewardID].ID
dailyReward[dailyRewardID].assetID
dailyReward[dailyRewardID].type
dailyReward[dailyRewardID].gender
dailyReward[dailyRewardID].data
dailyReward[dailyRewardID].metadata
dailyReward[dailyRewardID].name

Data properties:

dailyReward[dailyRewardID].data.resetDayIndex
dailyReward[dailyRewardID].data.days
dailyReward[dailyRewardID].data.name

There is only one object in dailyReward, which is

http://localhost:4000/v1/dailyReward/0

Which is the same as the default endpoint, so there is no need to simplify.
