Emote endpoint

The Emote endpoint gives info on the new facial expressions you can do in Prodigy.

Endpoint:

http://localhost:4000/v1/emote

Example request:

fetch('http://localhost:4000/v1/emote/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

emote[emoteID].ID
emote[emoteID].assetID
emote[emoteID].type
emote[emoteID].gender
emote[emoteID].data
emote[emoteID].metadata
emote[emoteID].name

Data properties:

emote[emoteID].data.name
emote[emoteID].data.isAnimated

Get more specific results using the Emote ID in the API endpoint.

http://localhost:4000/v1/emote/ID/

Example request:

fetch('http://localhost:4000/v1/emote/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":1150,"type":"emote","gender":0,"data":{"name":"Happy","isAnimated":1},"metadata":{"vIcon":2,"type":"multiple","assets":{"default":{"type":"atlas","v":1,"offSet":{"x":-1,"y":-2.5}}},"v":1},"name":"Happy"}
