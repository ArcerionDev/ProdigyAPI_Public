SkinColor endpoint

The SkinColor endpoint gives info on all of the skin tones you can have in the game.

Endpoint:

http://localhost:4000/v1/skinColor

Example request:

fetch('http://localhost:4000/v1/skinColor/').then(response => response.json()).then(data => {console.log(data)})

Example response:

Object properties:

skinColor[skinColorID].ID
skinColor[skinColorID].assetID
skinColor[skinColorID].type
skinColor[skinColorID].gender
skinColor[skinColorID].data
skinColor[skinColorID].metadata
skinColor[skinColorID].name

Data properties:

skinColor[skinColorID].data.price
skinColor[skinColorID].data.colors
skinColor[skinColorID].data.name
skinColor[skinColorID].data.flavorText

Get more specific results using the SkinColor ID in the API endpoint.

http://localhost:4000/v1/skinColor/ID/

Example request:

fetch('http://localhost:4000/v1/skinColor/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":360,"type":"skinColor","gender":0,"data":{"price":50,"colors":[[200,148,102],[255,235,210]],"name":"Skin Tone 1","flavorText":"Buying this will change your skin tone!"},"metadata":{"faceColor":1,"v":1,"type":"singleImage","vIcon":1},"name":""}
