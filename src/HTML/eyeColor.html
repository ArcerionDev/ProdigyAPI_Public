eyeColor endpoint

The eyeColor endpoint gets info on all of the purchaseable eye colors.

Endpoint:

http://localhost:4000/v1/eyeColor

Example request:

fetch('http://localhost:4000/v1/eyeColor/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

eyeColor[eyeColorID].ID
eyeColor[eyeColorID].assetID
eyeColor[eyeColorID].type
eyeColor[eyeColorID].gender
eyeColor[eyeColorID].data
eyeColor[eyeColorID].metadata
eyeColor[eyeColorID].name

Data properties:

eyeColor[eyeColorID].data.name
eyeColor[eyeColorID].data.price
eyeColor[eyeColorID].data.colors
eyeColor[eyeColorID].data.member
eyeColor[eyeColorID].data.flavorText

Get more specific results using the eyeColor ID in the API endpoint.

http://localhost:4000/v1/eyeColor/ID/

Example request:

fetch('http://localhost:4000/v1/eyeColor/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":345,"type":"eyeColor","gender":0,"data":{"name":"Light Brown","price":250,"colors":[[172,122,98]],"member":false,"flavorText":"Buying this will change your eye color!"},"metadata":{},"name":"Eye Color"}
