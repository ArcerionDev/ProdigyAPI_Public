Pet endpoint

The Pet endpoint gives info on all of the pets you can catch ingame... and even the ones you can't.

Endpoint:

http://localhost:4000/v1/pet

Example request:

fetch('http://localhost:4000/v1/pet/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

pet[petID].ID
pet[petID].assetID
pet[petID].type
pet[petID].gender
pet[petID].data
pet[petID].metadata
pet[petID].name

Data properties:

pet[petID].data.life
pet[petID].data.ordinal
pet[petID].data.name
pet[petID].data.curve
pet[petID].data.power
pet[petID].data.growth
pet[petID].data.rarity
pet[petID].data.element
pet[petID].data.flavorText
pet[petID].data.effects
pet[petID].data.price
pet[petID].data.statHealth
pet[petID].data.statPower
pet[petID].data.foreignSpellPools
pet[petID].data.nativeSpells
pet[petID].data.assetType

Get more specific results using the Pet ID in the API endpoint.

http://localhost:4000/v1/pet/ID/

Example request:

fetch('http://localhost:4000/v1/pet/0/').then(response => response.json()).then(data => {console.log(data)})
