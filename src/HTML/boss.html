Boss endpoint

Overview

The boss endpoint gives data about the numerous bosses in Prodigy.

Endpoint:

http://localhost:4000/v1/boss/

Example request:

fetch('http://localhost:4000/v1/boss/').then(response => response.json()).then(data => {console.log(data)})

Object properties:

bosses[bossID].ID
bosses[bossID].assetID
bosses[bossID].type
bosses[bossID].gender
bosses[bossID].data
bosses[bossID].metadata
bosses[bossID].name

Data properties:

- N/A

Get more specific results using the boss ID in the API endpoint.

http://localhost:4000/v1/boss/ID/

Example request:

fetch('http://localhost:4000/v1/boss/0/').then(response => response.json()).then(data => {console.log(data)})

Example response:

{"ID":1,"assetID":1863,"type":"boss","gender":0,"data":{},"metadata":{"assets":{"type":"multiple","default":{"type":"spritesheet","x":264,"w":512,"h":453,"v":5}}},"name":"Grandoff"}
